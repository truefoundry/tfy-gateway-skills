# Support

## Getting Help

For usage questions and troubleshooting:
- Open a GitHub issue in this repository.
- Include the skill name, exact command/query, and relevant logs/output.

For enterprise customers:
- Use your existing TrueFoundry support channel for production-impacting issues.

## Before Filing

Please include:
- Agent type (Codex, Claude Code, Cursor, etc.)
- OS and shell
- Relevant environment setup (`TFY_BASE_URL`, whether `TFY_API_KEY` is set)
- Steps to reproduce
- Expected behavior vs actual behavior

## Security Issues

Do not report security vulnerabilities in public issues. See `SECURITY.md`.
